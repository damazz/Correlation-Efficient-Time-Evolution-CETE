# Correlation-Efficient-Time-Evolution-CETE

This repository contains an implementation of the correlation-efficient time-evolution (CETE) algorithm. 

- Main implementation: src/CETE

- Exapmple notebook: notebooks/H2.ipynb

- Supporting code: src/CETE/CETE_helper_H2.py

- Dependencies: pyproject.toml

To use:

1) Clone github repository: 

    git clone git@github.com:damazz/Correlation-Efficient-Time-Evolution-CETE.git

2) Install and load dependencies

    pip install . 

3) Open notebook
